# TWRP Samsung Galaxy A31

## How to compile it:

To build:
export ALLOW_MISSING_DEPENDENCIES=true

setup envirnoment
`. build/envsetup.sh`

then prepare
`lunch twrp_a31-eng`

make the image
`mka recoveryimage`

# How to find the image built

`cd /out/target/product/a31`

see recovery.img
